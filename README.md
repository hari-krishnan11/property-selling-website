🏠 Real Estate Management System

## 📌 Description
This is my BCA Final Year Project. It is a web-based application for managing property listings, buyers, and sellers.

- Property Listing  
- Booking System  
- Admin Dashboard  
- Complaint Management  

## 🛠️ Technologies Used

- SQLite / MySQL  
- HTML, CSS, JavaScript  

## 📂 Modules

- Seller
